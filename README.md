* System dependencies
  * Bootstrap
  * React
  * Webpacker
  * Devise
  * *Pusher*
    * https://pusher.com/tutorials/ride-hailing-react-native

* Configuration
  * `rails s` to run server locally, *follow database create and initialization instructions*

* Database creation
  * `rails db:create`

* Database initialization
